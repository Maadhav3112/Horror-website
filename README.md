# BLACKWOOD ASYLUM

A horror-themed front-end website built using HTML, CSS, JavaScript and SVG interactions.

---

## Project Structure

horror-site/

├── index.html
├── map.html

├── css/
│ ├── style.css
│ ├── map.css
│ └── fonts.css

├── js/
│ ├── main.js
│ ├── map.js
│ ├── effects.js
│ └── utils.js

├── assets/
│ ├── asylum-map.svg
│ ├── fonts/
│ └── audio/

└── README.md

---

## Features

* Horror landing page
* Interactive asylum map
* Dynamic visual effects
* Screen shake effects
* Dark UI styling
* Modular folder structure
* Reusable CSS and JS

---

## Installation

Open terminal:

```bash
cd horror-site
```

Run local server:

```bash
python -m http.server 8000
```

Open:

```text
http://localhost:8000
```
